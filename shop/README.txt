How to Edit the Shop:
     1. Log in to customitsupport.com/shop/admin.php (use admin credentials).
     2. Products: Click product names to edit inline or use the form to add/delete.
     3. Visuals: Update colors/fonts in admin panel (saves to /shop/config.css).
     4. Images: Upload to /shop/images/ via admin panel or InfinityFree File Manager.
     5. Settings: Edit site name/tax in admin panel (saves to /shop/data.json).
     6. Backup: Edit /shop/data.json or /shop/config.css on GitHub (https://github.com/yourusername/customitsupport-shop) for version control.
     7. Contact: [Your contact] for help.
